再见

发表于 2017-04-08   |   分类于 随笔

看着同学在纸上写着年龄22的时候，我的脑袋白了一下。不对，走过这一年，我可以说走了两轮了吧，一直把自己当孩子，但却挡不出时间的洪流。照镜子的时候，发现脸上其实已经开始长细纹了。

曾以为自己早就习惯了一个人，无论是高中周五放学后空无一人的宿舍，还是现在周末空荡而乱七八糟的寝室，我好像安然地度过很多这样的日子，但是偶尔也会有寒流侵蚀入骨之感。总觉得一条路越走越孤寂，走着走着，一些人就要转弯了，走着走着，一些人就奔向别处了，有时你很想找个人好好说说话，然后看看周围，觉得不适合，还是太突兀，最后还是把那些话默默地吞到了肚子里。就这样，变得越来越沉默。

虽然过了那段间歇性发呆的日子，但每过一段时间，总觉得有什么事压着自己，很想大哭一场，却发现一滴眼泪也流不出来。
我好像知道什么事该做，什么事不该做，但令人懊恼的是，我失控了。明知道该面对的事，我在逃避，明知道该放一边的事，又固执地不肯停下来。这一周心情低落，做了一些错事，所以晚上有点辗转反侧，夜不能寐。我有点怀念高中的自己，那时候的自己似乎更强大一些。不知道到底是什么在不断提醒着自己已不再年轻，还有一些隐藏在深处不好的特性似乎慢慢浮出来，像是自私，妒忌，怯懦。我很害怕，更讨厌这样的自己。我恨自己不能洒脱，不能淡然，总是顾虑太多，不能说放就放。

清明和王出去走了走， 到一家寺院逛了逛，小家别院的感觉，丁香四溢，配上两株开的正盛的海棠，狭窄的侧边小道因为这两侧生动的丁香树变得有趣。游客来似乎是为了赏花，而不是拜佛的。我权当散心，简单拜拜。后来还去了牛街，有点想念朱，隔天吃饭的时候还听说朱被客户弄去一手机，觉得工作真不容易，廖说真是什么人都遇见了。我希望我们以后都能遇到善良的彼此，屏蔽掉那些丑恶。王说，这一宿舍的人都单着了。有点诧异，仍对未来一片迷茫。

七宗罪的结尾，年轻的探长还是被凶手激怒而开了枪，我讨厌失控的感觉，不想自己一错再错。
最近听的一首歌，词不达意，就如这篇随笔，文不对题。
